# Training Rules

These are the initial coaching rules used by the local running coach.

## Readiness Logic
- If form > +10, allow race or hard workout.
- If form between -5 and -15, threshold workout allowed.
- If form < -20, easy run only.
- Otherwise, default to a steady aerobic session.

## Coaching Principles
- Keep training progression sustainable.
- Avoid assigning hard sessions when readiness is low.
- Prefer simple, explainable choices over complex scoring systems.
- Keep workouts readable and structured.
